# Spécifications Fonctionnelles : V0.1.5 — Stratégie 12WY 🧿

## S-12WY-01 : Grille des 12 Semaines
**User Story** : En tant que stratège, je veux voir mon cycle de 12 semaines divisé en cartes interactives afin de suivre l'avancement de mon sprint de vie.
### Acceptance Criteria
- [ ] Grille latérale (Gauche: W1-W6, Droite: W7-W12).
- [ ] Chaque carte affiche : Numéro de semaine, % d'avancement, Mini-graphique de tendance.
- [ ] Statuts : Complete (Passé), Active (Courant), Upcoming (Futur).

## S-12WY-02 : Objectifs Tactiques Centraux
**User Story** : En tant qu'exécuteur, je veux voir mes objectifs tactiques prioritaires au centre de ma page 12WY.
### Acceptance Criteria
- [ ] Section centrale "Tactical Objectives".
- [ ] 3-4 Cartes d'objectifs avec jauges de progression circulaires (ex: "Solar Project 70%").
- [ ] Connexion avec les tâches PARA (LD01) filtrées par horizon 12WY.

## S-12WY-03 : Header Menu de Navigation (Strategy)
**User Story** : En tant qu'utilisateur, je veux basculer entre les différentes phases de planification.
### Acceptance Criteria
- [ ] Toolbar supérieure : "VISION", "PLANNING", "PROCESS CONTROL", "MEASUREMENT", "TIME USE".
- [ ] Chaque bouton redirige vers une sous-vue filtrée de 12WY.
