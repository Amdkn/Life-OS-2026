# Design : V0.1.7 — DEAL Liberation Engine 🧿

## Architecture
Le `DealApp` est un orchestrateur de 4 modules indépendants connectés à LD08 :
- `MuseReactorView` (Noyau central).
- `TmiCalculator` (Top Bar).
- `AutomationGraph` (Module complexe utilisant une lib de graphe type React Flow).
- `FreedomMap` (Visualisation géographique).

## Esthétique (High-Tech Solarpunk)
Le réacteur utilise des shaders de luminescence.
Les connexions entre les modules simulent un circuit imprimé organique.
Typographie technique (`JetBrains Mono` ou `Orbitron`) mélangée à `Outfit` pour le contraste Archo-Futuriste.
