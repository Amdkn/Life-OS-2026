# Design : V0.1.3 — Ikigai Module View 🧿

## Layout de la Page
Double conteneur principal :
1. `IkigaiModule` (Upper) : Layout 3 colonnes (Passions / Diagramme / Missions) + Piliers inférieurs.
2. `TemporalHorizons` (Lower) : Slider ou Grille de 5 cartes horizontales.

## Système de Routage
Utilisation de `aspace://app/ikigai?view=horizon&id=h1` pour naviguer vers les spécificités d'un horizon.

## Esthétique
- Fond : Verre dépoli "Solarpunk" avec des nuances vertes et ocres.
- Cartes : Style "Neumorphic-Solar" avec des icônes cuivrées.
- Connecteurs : Lignes de lien fines reliant les piliers au diagramme central (simulant une croissance organique).
